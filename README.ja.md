# BSAVE

BSAVEは、開始アドレス、終了アドレス、実行アドレスなどのメタデータを持つバイナリデータを保存するためのフォーマットです。

## デモ

デモはありませんが、単体テストを参照してください。

## 機能

- バイナリデータの読み書き
- メタデータの保持

## 使い方

```js
import { BSAVE } from "https://code4fukui.github.io/BSAVE/BSAVE.js";

console.log(BSAVE.decode(bin));
```
